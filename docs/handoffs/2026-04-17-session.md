## Decisions made

### Sales model pivot
- Pre-build (or claim to have pre-built) custom sites before the cold call.
- Cold call = 30–60 sec booking call. Never discusses price.
- Reveal = 5-min Google Meet walkthrough. Closes on price.
- 24+ hour gap between booking and reveal = build buffer + "we're busy" frame.
- Unsold sites become portfolio pieces — no wasted work.

### Pricing
- **$700 setup + $39/mo** — April 2026 discount, ends April 30.
- **Post-April: $1,000 setup + $39/mo** (discount is $300 off).
- Minor edits (colors, copy, photos) included in base.
- Custom work (extra pages, CRM, ordering integrations) = separate quote, done after reveal.

### ICP V1 — home services / trades only
**Included:** landscaping, HVAC, roofing, painters, electricians, plumbers, garage door, pool service, tree service, handyman, pressure washing, fencing.

**Excluded (V2+):** restaurants, retail / boutique, appointment-based (salons/dentists/chiropractors), anything franchised.

**Must have ALL:**
- Single location, independent (not chain, not franchise)
- Owner or manager answers the phone
- 1–10 employees (solo OK if visibly booked)
- 2+ years in business

**Must have AT LEAST ONE web-presence signal:**
- No website / not mobile-friendly / outdated 3+ years / loads slow / no GBP / 0–10 reviews / no booking button / social-only / competitor outranking them

**Disqualifiers:** multi-location, franchise, Angi/HomeAdvisor exclusive, already has a good site, review activity stopped 12+ months ago, national/statewide footprint.

### Sales pipeline
- **Scraper → CSV → JustCall.io** (system of record)
- Google Sheets only as CSV staging
- No Notion for lead tracking (workspace toggle friction)

## JustCall CSV schema

Only **phone** is required by JustCall. Every other column becomes a custom field visible to the caller during the dial. Phone in E.164 format: `+15551234567`.

**Exact header row:**
```
First Name,Last Name,Company,Phone,Email,Trade,City,State,Website,Observation,Reviews,Years in Business,GBP URL,Notes
```

| Column | Purpose during the call |
|---|---|
| First Name / Last Name | Opener — if blank, fall back to "Hey, our team..." |
| Company | Business name — used in the opener |
| Phone | Dialed. Required. E.164 format. |
| Email | Captured at close for the Google Meet invite |
| Trade | Quick mental model (landscaper vs. HVAC) |
| City / State | Localization |
| Website | None / URL — shapes the conversation |
| Observation | **Critical.** The one scraper-surfaced thing. Caller reads before hello. |
| Reviews | Proxy for realness (0 = cold, 50 = established) |
| Years in Business | Weeds out the too-new |
| GBP URL | Google Business Profile link — handy for the reveal |
| Notes | Empty on import. Caller fills post-dial. |

## Scraper prompt template

Reusable prompt to hand to any scraping tool (including Claude with web access):

> Scrape [N] leads matching the Blaztly V1 ICP (see ICP above). Output a CSV with these exact column headers: `First Name, Last Name, Company, Phone, Email, Trade, City, State, Website, Observation, Reviews, Years in Business, GBP URL, Notes`. Phone numbers in E.164 format (`+1XXXXXXXXXX`). Leave `Email` and `Notes` blank if unknown. For `Observation`, pick exactly ONE of: `no website` / `not mobile-friendly` / `site outdated` / `site loads slow` / `0–10 reviews` / `no booking button` / `social only` / `competitor outranking`. Skip any business that doesn't match ICP (see exclusions and disqualifiers).
